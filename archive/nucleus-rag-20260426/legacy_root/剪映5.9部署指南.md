# 剪映 5.9 部署指南

## 为什么要锁定 5.9

剪映 6.0 以上版本对草稿文件（draft_content.json）做了加密，导致我们无法读取剪辑师修改后的草稿内容。5.9 版本的草稿是明文 JSON，可以实现：

- AI 自动生成草稿（含素材推荐）
- 剪辑师在剪映中编辑
- 系统自动对比修改前后的差异，收集反馈，让素材推荐越来越准

**锁定 5.9 是整个素材反馈系统的基础，请勿升级。**

---

## 安装步骤

### 1. 卸载现有剪映

如果电脑上已安装剪映 6.0+，先卸载：
- 打开 Windows 设置 → 应用 → 找到"剪映专业版" → 卸载

### 2. 安装剪映 5.9

安装剪映专业版 5.9.0 安装包（找组长获取）。

### 3. 安装完成后，立刻运行防更新脚本

**关闭剪映**，然后以管理员身份打开 PowerShell，运行以下脚本：

```powershell
$appDir = "$env:LOCALAPPDATA\JianyingPro\Apps"
$verDir = Get-ChildItem $appDir -Directory | Where-Object { $_.Name -match '^\d' } | Select-Object -First 1

# 1. 改名所有更新程序
foreach ($name in @("update.exe", "JianyingPro-DiffUpgrade.exe")) {
    $f = Join-Path $verDir.FullName $name
    if (Test-Path $f) {
        Rename-Item $f "$name.disabled" -Force
        Write-Host "已禁用: $name"
    }
}

# 2. 创建同名空文件夹占位（防止重新下载）
foreach ($name in @("update.exe", "JianyingPro-DiffUpgrade.exe")) {
    $f = Join-Path $verDir.FullName $name
    if (-not (Test-Path $f)) {
        New-Item -ItemType Directory -Path $f -Force | Out-Null
        Write-Host "已占位: $name（文件夹）"
    }
}

# 3. 清理更新缓存
$updateInfo = "$env:LOCALAPPDATA\JianyingPro\User Data\Config\updateInfo"
if (Test-Path $updateInfo) {
    Remove-Item $updateInfo -Recurse -Force
    New-Item -ItemType Directory -Path $updateInfo -Force | Out-Null
    $acl = Get-Acl $updateInfo
    $rule = New-Object System.Security.AccessControl.FileSystemAccessRule(
        $env:USERNAME, "Write", "Deny")
    $acl.AddAccessRule($rule)
    Set-Acl $updateInfo $acl
    Write-Host "已锁定: updateInfo 目录"
}

# 4. 用 hosts 屏蔽剪映更新服务器（需要管理员权限）
$hostsPath = "$env:SystemRoot\System32\drivers\etc\hosts"
$blockDomains = @(
    "lf-package-cdn.jianying.com",
    "lf-package.jianying.com"
)
$hostsContent = Get-Content $hostsPath -Raw
foreach ($domain in $blockDomains) {
    if ($hostsContent -notmatch [regex]::Escape($domain)) {
        Add-Content $hostsPath "`n127.0.0.1 $domain"
        Write-Host "已屏蔽: $domain"
    }
}

Write-Host "`n完成！请重新打开剪映确认版本号为 5.9.x"
```

### 4. 断网验证

1. 打开剪映，确认左上角版本号显示 **5.9.x**
2. 如果弹出升级提示，直接关掉
3. 使用一段时间后再次确认版本号没有变化

## 注意事项

- **不要点击任何升级/更新按钮**，如果偶尔弹出提示，直接关掉
- **不要重新安装剪映**，如果需要重装，找组长获取 5.9 安装包并重新运行防更新脚本
- 如果发现版本号变了，说明更新没挡住，联系组长重新部署
- 草稿文件位置：`C:\Users\你的用户名\AppData\Local\JianyingPro\User Data\Projects\com.lveditor.draft\`
- **重要：必须断开剪映的登录账号（不要登录剪映账号），防止云端触发强制更新**

## 常见问题

**Q：5.9 缺少某些功能怎么办？**
A：我们的字幕、AI 分析、渲染、素材推荐都是自己的系统生成的，不依赖剪映的 AI 功能。5.9 的基础剪辑功能完全够用。

**Q：我不小心升级了怎么办？**
A：卸载后重新安装 5.9，按上面的步骤操作。草稿文件不会丢失。

**Q：能打开别人发来的草稿吗？**
A：可以。我们系统生成的草稿是明文 JSON，5.9 可以正常打开。
