## 一句话定位

原始视频 → AI 三级语义分析 → Remotion 渲染 overlay → 剪映草稿自动生成（含素材推荐）→ 分发到剪辑师电脑 → 剪辑师微调 → 自动收集反馈。

系统边界：只做 overlay 信息图形 + 素材推荐 + 草稿生成，不做人像抠图和最终合成。最终合成由剪辑师在剪映中完成。

## 整体架构

### 角色分工

| | 剪辑服务器 | 剪辑师电脑（6台） |
|--|--|--|
| AI 管线 | 跑 analyze / render / 草稿生成 | — |
| 素材库 | 服务器本地共享文件夹（千兆局域网） | 映射统一盘符访问 |
| 草稿分发 | 直接写到 `\\剪辑师电脑\com.lveditor.draft\` | 打开剪映即可使用 |
| 草稿回收 | 从同一网络路径读回修改后的草稿 | 剪辑师正常保存即可 |
| diff 分析 | 自动对比原始 vs 修改后草稿 | — |
| 剪映版本 | — | 锁定 5.9（明文草稿） |

### 数据流

```
音频 → Qwen3-ASR 转录 → Step1 语义拆解 → Take-Pass keep/discard
  → Step2 结构编排 + 素材匹配(asset_clip) → Blueprint
  → Timing Map → Remotion 渲染 overlay → SRT 导出
  → 生成剪映草稿(含素材轨道) → 写到剪辑师电脑
  → 剪辑师编辑 → diff-draft.py 对比 → 反馈入库
```

### 核心架构

```
服务器生成草稿 → 写到剪辑师电脑 → 剪辑师正常剪辑 → 服务器读回草稿 → diff → 反馈
     ↑                                                                    ↓
     ←←←←←←←←←←← 更新素材索引 / 偏好权重 ←←←←←←←←←←←←←←←←←←←←←←←←←←←←←←
```

### 反馈数据来源（全自动，剪辑师零额外操作）

- 素材被删除 → 该素材与该语境的匹配分 -1
- 素材被保留 → 匹配分 +1
- 素材被替换 → 新素材与该语境建立关联
- 新增素材（不在索引中）→ 收集文件路径，定期批量打标入库
- 音频被删除 → 全局规则：素材轨默认静音

## 八、完整 E2E 命令

```powershell
$dir = "P:/团队空间/.../xxx kp01"
$out = "$dir/out"
$mp4 = (Get-ChildItem "$dir/*.mp4" | Select-Object -First 1).FullName
$docx = (Get-ChildItem "$dir/*.docx" | Select-Object -First 1).FullName

# 1. Analyze（转录 + 语义分析 + 素材匹配）
npm run analyze -- --audio "$mp4" --script "$docx" -o "$out/blueprint.json" --transcribe-qwen --force-align-qwen --asset-index "asset_index.json"

# 2. Timing
npm run timing:direct -- --input "$mp4" -b "$out/blueprint.json" -o "$out/timing_map.json"

# 3. Render
npm run render -- -b "$out/blueprint.json" -t "$out/timing_map.json" --source-video "$mp4" -o "$out/overlay.mp4"

# 4. SRT
npx tsx src/renderer/export-srt.ts -b "$out/blueprint.json" -t "$out/timing_map.json" -o "$out/subtitles.srt"

# 5. 生成草稿（直接写到剪辑师电脑）
python scripts/generate-jianying-draft.py "$out" --asset-index "asset_index.json" --target "\\剪辑师电脑\com.lveditor.draft"

# 6. 剪辑师编辑后，对比差异
python scripts/diff-draft.py "$out/xxx_draft/draft_content.json" "\\剪辑师电脑\com.lveditor.draft\xxx_draft\draft_content.json"
```
