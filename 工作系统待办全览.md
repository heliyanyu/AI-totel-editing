## 五、完整 E2E 命令（API Key 模式）

```powershell
$dir = "P:/团队空间/公司通用/AIkaifa/AI total editing/260323/xxx kp01"
$out = "$dir/out"
$mp4 = (Get-ChildItem "$dir/*.mp4" | Select-Object -First 1).FullName
$docx = (Get-ChildItem "$dir/*.docx" | Select-Object -First 1).FullName

# 1. Analyze（转录 + 语义分析）
npm run analyze -- --audio "$mp4" --script "$docx" -o "$out/blueprint.json" --transcribe-qwen --force-align-qwen

# 2. Timing
npm run timing:direct -- --input "$mp4" -b "$out/blueprint.json" -o "$out/timing_map.json"

# 3. Render（overlay + source_direct_cut_video）
npm run render -- -b "$out/blueprint.json" -t "$out/timing_map.json" --source-video "$mp4" -o "$out/overlay.mp4"

# 4. SRT
npx tsx src/renderer/export-srt.ts -b "$out/blueprint.json" -t "$out/timing_map.json" -o "$out/subtitles.srt"

# 5. 编辑器
npm run editor:serve -- "$out"
```

**断点续跑（已有 reviewed_tokens.json，跳过转录和 review）：**
```powershell
npm run analyze -- --transcript "$out/transcript_aligned_tokens.json" --skip-review -o "$out/blueprint.json"
```
